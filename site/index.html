Quick start

From template to first page in minutes.

# Scaffold a new project from the template
npm create astro@latest my-site -- --template JUNGHERZ/GlassKit-Web
cd my-site
npm install
npm run dev

Or click “Use this template” on GitHub – then follow the recipe in SKILL.md, or hand that file to your AI assistant and let it do the deriving — the ready-made prompt is in the docs.
