Quick Start

Vom Template zur ersten Seite in Minuten.

# Neues Projekt aus dem Template aufsetzen
npm create astro@latest meine-site -- --template JUNGHERZ/GlassKit-Web
cd meine-site
npm install
npm run dev

Oder auf GitHub auf „Use this template" klicken – danach dem Rezept in der SKILL.md folgen, oder die Datei dem KI-Assistenten geben und das Ableiten ihm überlassen — der fertige Prompt steht in der Doku.
